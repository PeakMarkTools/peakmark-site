FAQ

Common Questions.

What operating systems does PeakMark support?
PeakMark is a Windows application built for Windows 10 and Windows 11 (x64). It is installed as a native desktop app via a standard Windows installer (.exe). Mac and Linux support is not currently planned.
What is a "seat" and how does seat licensing work?
A seat is one activated machine. When PeakMark is installed on a computer and the license key is entered, that machine occupies one seat. You can deactivate a machine to free its seat and reassign it to another. Starter allows 5 simultaneous machines, Pro allows 25, and Enterprise is unlimited.
How does the auto-updater work?
PeakMark checks for updates automatically every time the app launches. When a new version is available, a notification appears in the app. Your tech can download and install the update in one click without leaving the app. Updates are hosted on GitHub Releases and delivered securely.
Can I customize which tests run per client?
Yes. Each client profile comes with a preset selection of tests designed for that use case, but you can add additional optional tests on top of the preset before running. Nothing is locked — the profile is a smart starting point, not a restriction.
Is PeakTune safe? Can it damage hardware?
PeakTune only surfaces optimizations that have been stability-validated. It does not write any BIOS settings, does not force overclocks, and does not make any changes without your technician clicking "Apply." Recommendations that require BIOS changes (like enabling XMP/EXPO) are flagged as manual steps with instructions, never applied automatically.
What does the early access offer include?
Waitlist members who sign up before launch receive 30 days free on any plan — no credit card required for the trial period. You'll receive an email with your license key and download link on launch day.
Who makes PeakMark?
PeakMark is developed by PeakMark LLC, a subsidiary of Apex Property Ventures, based in Wake Forest, NC. The software was built by technicians for technicians — designed around real machines and real shop workflows.
